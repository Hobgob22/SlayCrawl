API Usage

                
To use the API programmatically, send a POST request to /scrape with your API key. You can get your API key from the API Keys page.

                
curl -X POST "http://localhost:8000/scrape" \
    -H "X-API-Key: your-api-key" \
    -H "Content-Type: application/json" \
    -d '{"url": "https://example.com", "render_js": false}'
